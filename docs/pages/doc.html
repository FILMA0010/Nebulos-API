Get Started

API Endpoints

Use Image API

              
                GET https://api.nebulos.pro:3000/api/images/${category}

Get Image API categories

              
                GET https://api.nebulos.pro:3000/api/images/categories

Use Text API

              
                GET https://api.nebulos.pro:3000/api/text/${category}

Get Text API Categories

              
                GET https://api.nebulos.pro:3000/api/text/categories

Use Roblox API

              
                GET https://api.nebulos.pro:3000/api/roblox/{category}/${id}

Get Roblox API Categories

              
                GET https://api.nebulos.pro:3000/api/roblox/categories

Use Minecraft Server API

              
                GET https://api.nebulos.pro:3000/api/minecraft/server/${version}/${adress}

Use Minecraft User API

              
                GET https://api.nebulos.pro:3000/api/minecraft/${uuid}

Get Minecraft API Categories

              
                GET https://api.nebulos.pro:3000/api/minecraft/categories

API Values

Value Description Required
${category} Category of the API true
${id} ID of user/group true
${adress} Adress [Domain or IP Adress] true
${version} Minecraft Bedrock/Java true
${uuid} Minecraft UUID true
